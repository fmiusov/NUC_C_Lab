| vi Keystrokes      | Description |
| ----------- | ----------- |
|ZZ|save and quit|
|:e!|remove all current edits, return to original session|
|:q!|remove all current edits, close vi|
|:w!|overwrite existing file|
|Esc|enter command mode / beep mode|
|i|enter insert mode|
